Advanced / Project-Based Team

Team Officers: Eeshwar, Abhiram, and Owen

Team members will be more focused on project-based learning, and will learn specific skills and programming topics in order to progress their knowledge. Topics and activities will be considered as an AP or higher skill level. Programming topics may include AI, web development, game development, IOS development, and more, covering a wide variety of coding languages.

Fundamental Team

Team Officers: Sanah, Rohan, and Aashi

Team members will learn basic programming skills in order for them to accelerate their programming knowledge. Programming topics will include basic to semi advanced coding skills such as but not limited to data types, data structures, arrays, algorithms, classes, objects, loops, if statements, console input, file input, etc.
