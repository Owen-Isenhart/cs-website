JAVA

  • Java is the 3rd most popular programming language, used for Andriod app development, desktop GUI applications, game development, and big data technology. If you're taking programming classes at FHS, this is the language you'll be using and learn.

    BlueJ or IntelliJ are the recommended IDEs for this language.

PYTHON

  • Python is the most popular programming language due to it being easy to learn and it's ability to be used for many things. Most commonly, it's used for Data analytics, machine learning, web development, and financial analysis.

    PyCharm, VS Code, or Atom are the recommended IDEs for this language.

C#

  • C# is a very versatile and fast language, commonly used for game development. If you take Video Game Programming at the CTE Center, this is the language you'll use. It's very similar in syntax to Java, so if you know Java, you basically know C#.

    Rider or Visual Studio are the recommended IDEs for this language.

HTML + CSS

  • HTML and CSS are the languages used to structure and style websites. Everything you see on this website was built using solely HTML and CSS. HTML is what's used to create things on a website, and CSS can color, size, and position the things on the website.

    VS Code, or Atom are the recommended IDEs for this language.

SWIFT

  • Swift is Apple's relatively new programming language used for creating IOS apps. It has a vaguely similar syntax to Python, making it pretty easy to pick up if you have some experience. If you take Mobile Applications Programming at the CTE Center, this is what you'll be using.

    XCode is the recommended IDE for this language, but it requires a Mac.

JAVASCRIPT

  • JavaScript is a language that's used in tandem with HTML and CSS to make websites more interactive. While it's not needed to make simple websites (it wasn't used for this one), most websites with professional development will be using JavaScript.

    VS Code, or Atom are the recommended IDEs for this language.

C AND C++

  • Despite their names, these languages are actually quite different. C is often used for embedded devices and OS kernels, like Windows. C++ on the other hand is used for more graphic heavy software like games, photo/video editing apps, and browsers.

    Eclipse or VS Code are the recommended IDEs for this language.
